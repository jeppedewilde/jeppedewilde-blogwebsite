Rosa kwam een heel ander verhaal vertellen: over de infrastructuur van het web en hoe we minder afhankelijk kunnen worden van Big Tech.

## Onderwerpen & Tools
Ze had het over **Alternative Infrastructure** en het zelf hosten van je eigen server. Waarom alles in 'de cloud' (wat gewoon andermans computer is) zetten als het ook op een Raspberry Pi thuis kan?

**Kernpunten:**
- **DIY / DIWO:** Do It Yourself / Do It With Others.
- **Static Site Generators:** Zoals Astro, het heeft minder impact op een server en is heel robuust.
- **Low-tech:** Gebruik van oude hardware dwingt je om beter na te denken over je code en performance.

## Wat ga ik meenemen?
Ik vind het idee van een 'travelling server' of een eigen Raspberry Pi server heel vet. Ik wil me meer gaan verdiepen in hoe ik mijn eigen data weer 'van mij' kan maken.
